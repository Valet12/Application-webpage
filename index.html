My Projects

Artificial Intelligent Self-Driving Robot Car

Built an AI system that reads letters from images, computes the shortest driving path, and sends coordinates to a microcontroller to autonomously navigate a robot car.

Interactive Hand Strength Rehabilitation System

Paired a Velostat-embedded ceramic grip with Unity and Arduino to measure squeeze pressure, streaming real-time feedback into a game that makes hand-strength exercises engaging.

Game Using Machine Vision and IMU Sensor

Fused HebiCam (MATLAB) and smartphone IMU data to capture real-time movements, translating them into Unity space-game controls for smooth, responsive gameplay.

VR Game with Custom Arduino Hand Controller

Designed and built a DIY Arduino hand controller (Xbox-style) with buttons and joysticks, then integrated it into a Unity-based VR game to enable intuitive, tactile interaction and navigation in virtual environments.

AI Path Generator with Random Points

Created a MATLAB app where users place random points and a Hopfield network computes an efficient route through all points, demonstrating AI for solving combinatorial optimization.
